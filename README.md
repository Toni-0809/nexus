# Nexus

Десктопное приложение для управления записями.

## Описание проекта

Nexus — это локальное desktop-приложение для хранения, редактирования и структурирования записей в виде страниц и блоков.  
Приложение поддерживает связи между блоками, поиск, автосохранение, импорт и экспорт Markdown, визуальную карту связей и настройки интерфейса.

## Основные возможности

- создание, выбор, переименование и удаление страниц
- добавление, редактирование и удаление блоков
- перестановка блоков с помощью drag & drop
- создание связей между блоками
- отображение обратных ссылок (backlinks)
- поиск по страницам и блокам
- автосохранение изменений
- экспорт записей в Markdown
- импорт записей из Markdown
- визуальная карта связей текущей записи
- настройка темы, шрифта, размера текста и акцентного цвета

## Используемые технологии

- Java 21
- JavaFX 21
- Maven
- SQLite
- JDBC Driver
- JavaFX CSS
- Preferences API


## Порядок запуска
Порядок запуска проекта

1. Установить Java 21.
2. Проверить установку командой:
   java -version

3. Установить Apache Maven.
4. Проверить установку командой:
   mvn -version

5. Открыть терминал в папке проекта, где находится файл pom.xml.
6. Выполнить сборку проекта командой:
   mvn clean compile

7. Запустить приложение командой:
   mvn javafx:run

8. После запуска откроется окно приложения Nexus.


## База данных

- При первом запуске приложение автоматически создаёт локальный файл базы данных nexus.db.
- Если файл nexus.db уже находится в папке проекта, приложение откроет сохранённые ранее записи.

В базе данных используются три основные таблицы:

* `pages`
* `blocks`
* `links`
